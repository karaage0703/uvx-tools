# hello-uvx

A simple hello world tool for demonstrating uvx.

## Usage

```bash
uvx --from https://github.com/karaage0703/uvx-tools hello-uvx
uvx --from https://github.com/karaage0703/uvx-tools hello-uvx "Your Name"
```
